Python for Data Science

The essential quick-reference guide for NumPy, Pandas, and Matplotlib.

1. NumPy (Numerical Python)

NumPy is the fundamental package for scientific computing. It provides a high-performance multidimensional array object.

PYTHON
import numpy as np # Creating Arrays arr = np.array([1, 2, 3]) # 1D Array mat = np.zeros((3,3)) # 3x3 Zero Matrix rand = np.random.rand(2,2) # Random values 0-1 # Inspections shape = arr.shape # Dimensions (rows, cols) dtype = arr.dtype # Data type (int64, float32) # Math Operations prod = np.dot(a, b) # Matrix Multiplication mean = np.mean(arr) # Average
⚡ Pro Tip: Always prefer NumPy vectorized operations (like arr * 2) over Python loops. They are implemented in C and are 100x faster.

2. Pandas (Data Analysis)

Pandas is built on top of NumPy and provides easy-to-use data structures and data analysis tools.

Dataframes & Loading

PYTHON
import pandas as pd # Loading Data df = pd.read_csv('data.csv') df = pd.read_excel('sheet.xlsx') # Viewing Data df.head(5) # First 5 rows df.info() # Index, Datatype and Memory info df.describe() # Summary statistics (mean, std, min)

Cleaning Data

PYTHON
# Handling Missing Values df.dropna() # Drop rows with nulls df.fillna('Unknown') # Fill nulls with value # Filtering adults = df[df['age'] > 18] # Filter rows subset = df[['name', 'age']] # Select columns

3. Matplotlib (Visualization)

PYTHON
import matplotlib.pyplot as plt x = [1, 2, 3, 4] y = [10, 20, 25, 30] plt.plot(x, y, color='red', linestyle='--') plt.title("Growth Over Time") plt.xlabel("Days") plt.ylabel("Revenue") plt.show()

1. Pandas (Data Manipulation)

Importing & Loading

PYTHON
import pandas as pd

# Load data from CSV
df = pd.read_csv('dataset.csv')
# View first 5 rows
df.head()

Inspecting Data

PYTHON
df.info() # Check data types & missing values
df.describe() # Summary statistics (mean, max, etc.)
df.shape # (rows, columns)

Selection & Filtering

PYTHON
# Select column
ages = df['Age']

# Filter rows where Age > 25
adults = df[df['Age'] > 25]

# Select by Label (loc) vs Index (iloc)
row = df.loc[5] # Row with index label 5
row = df.iloc[0] # First row (index 0)

2. NumPy (Numerical Computing)

Arrays & Math

PYTHON
import numpy as np

# Create Arrays
arr = np.array([1, 2, 3])
zeros = np.zeros((3,3)) # 3x3 matrix of zeros

# Operations
mean_val = np.mean(arr)
max_val = np.max(arr)
reshaped = arr.reshape(3,1)

3. Matplotlib (Visualization)

Basic Plotting

PYTHON
import matplotlib.pyplot as plt

x = [1, 2, 3]
y = [10, 20, 30]

plt.plot(x, y, color='red')
plt.title('Sales Growth')
plt.xlabel('Month')
plt.show()

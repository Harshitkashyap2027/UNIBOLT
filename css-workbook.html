Mastering Layouts: Flexbox & Grid

A visual guide to modern CSS layout techniques.

1. Flexbox (One-Dimensional)

Use Flexbox for alignment within a single row or column (e.g., Navbars, Centers).

CSS
.container { display: flex; justify-content: space-between; /* Horizontal Alignment */ align-items: center; /* Vertical Alignment */ gap: 20px; /* Spacing */ }
💡 Cheat Sheet:
justify-content = Main Axis (Left/Right)
align-items = Cross Axis (Up/Down)

2. CSS Grid (Two-Dimensional)

Use Grid for overall page layouts involving both rows and columns.

CSS
.grid-layout { display: grid; /* Create 3 columns: 200px, auto-fill, and 1 fraction */ grid-template-columns: 200px auto 1fr; grid-template-rows: 100px auto; gap: 15px; }

2.1 The "Holy Grail" Layout

CSS
.page { display: grid; grid-template-areas: "header header" "sidebar main" "footer footer"; grid-template-columns: 250px 1fr; }
